CopperSpice

CopperSpice is a set of libraries written in C++. The libraries were derived from the Qt 4.8 framework in conjunction with the C++ Standard library.

The core libraries were redesigned leveraging template functionality and taking advantage of modern C++. CopperSpice consists of several different libraries to handle core functionality, GUI interfaces, networking, web capabilities, and much more.

  • CopperSpice libraries were inspired by the LGPL Qt libraries
  • CopperSpice was written in C++11 and then migrated to C++14
  • Our libraries can link directly into any modern C++ application


For additional information about the CopperSpice libraries please refer to our web site: CopperSpice
