Building

Diamond was compiled with the CopperSpice libraries using GCC (4.7.2 or greater), GNU Autotools, and GNU Make. To build Diamond from source all of the approprieate libraries and tools must be installed on your development system.

CopperSpice Libraries

The CopperSpice binary download contains the DLLs or shared libraries needed to compile and distribute Diamond.

  • (1)  Click on th link to be directed to the CopperSpice web site   CopperSpice
  • (2)  On the right side of the page is a 'downloads' box, lick on the desired version of CopperSpice
  • (3)  Select the binary folder and then click on the distribution or installer for your platform
  • (4)  Install the distribuiton or run the installer

Hunspell Libraries

Diamond uses the Hunspell 1.3 library for spell checking.

  • X11 - install the appropriate Hunspell package for your specific platform
Platform Package
Debian libhunspell-dev
Fedora hunspell-devel
Ubuntu libhunspell-dev

  • Windows

    • The 32-bit dll for Hunspell is included with the Diamond 32-bit binary installer
    • The 64-bit dll for Hunspell is included with the Diamond 64-bit binary installer
    • The Hunspell 1.3.2 source files can be downloaded from our repository, required to build Diamond from source
      Diamond Source
  • OS X - use macports to install Hunspell

Autotools & Make

Autotools is a suite of programming tools designed to assist in making source code packages portable to many Unix-like systems. The GNU build system uses Bourne-compatible shell scripts to assist the user in the configuration and build process.

The utilities used to build Diamond are listed below.

  • Autoconf version 2.67 or greater
  • Automake version 1.12.1 or greater
  • GNU Make version 3.81 or greater

For additional information about these utilities, refer to our CopperSpice documentation Building CopperSpice
